## FindJob

Сервис по поиску работы и размещению вакансий.

### Реализовано
- Регистрация и вход с ролями `seeker` (соискатель) и `company` (компания).
- Компании могут:
  - размещать вакансии;
  - просматривать отклики на свои вакансии;
  - просматривать резюме и отправлять приглашения кандидатам.
- Соискатели могут:
  - создавать/обновлять резюме;
  - откликаться на вакансии;
  - видеть свои отклики и приглашения от компаний;
  - писать отзывы о компаниях.
- Общий список отзывов о компаниях доступен на главной странице.

### Запуск
1. Установить зависимости:
   - `npm install`
2. Запустить API:
   - `npm run server`
3. В отдельном терминале запустить фронтенд:
   - `npm run dev`

### Демо-аккаунты
- Компания: `company@example.com` / `123456`
- Соискатель: `seeker@example.com` / `123456`
